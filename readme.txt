Simple Cookie Consent Management Platform with Google Consent Mode v2.

== Description ==

Custom Cookie CMP is a lightweight cookie consent plugin for WordPress that helps you comply with GDPR and other privacy regulations. It integrates with Google Consent Mode v2 and gives visitors control over which cookies they accept.

**Features:**

* Cookie consent banner with customizable text and colors
* Google Consent Mode v2 integration
* Configurable consent expiry
* Lightweight — no external dependencies
* WPML/Polylang ready (translation-ready)

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/custom-cookie-cmp/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to **Settings → Cookie CMP** to configure the banner

== Frequently Asked Questions ==

= Does this plugin support Google Consent Mode v2? =
Yes, it fully integrates with Google Consent Mode v2.

= Is it GDPR compliant? =
The plugin provides the UI tools needed for GDPR compliance. Final compliance depends on your site configuration.

= Can I customize the banner appearance? =
Yes, you can change colors, button text, and banner message from the settings page.
